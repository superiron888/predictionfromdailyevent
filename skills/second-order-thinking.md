# Second-Order Thinking — Expectation Gap Detection

> "The first-level thinker says: 'Good company, buy it.'
> The second-level thinker says: 'Good company, but everyone knows it, so overvalued, sell it.'"
> — Howard Marks

## Why This Matters
Butterfly chains are naturally first-order: Event → affects B → Industry C benefits.
But the market is an expectation machine. If you can deduce it, others can too → your reasoning = consensus = no edge.
**Mr.IF Spark's value is not "what will happen" but "what hasn't been priced in yet."**

## Three Levels
- **Level 0 (Fact)**: What happened?
- **Level 1 (First-order)**: So what? — Everyone's reaction
- **Level 2 (Second-order)**: But the market already knows, so...? — The expectation gap
- **Level 3 (Game theory)**: What do other players (Fed, institutions, short-sellers) think I think? Useful when positioning is extreme. Example: Everyone expects Fed to cut → bond market rallies pre-emptively → if Fed doesn't cut, violent reversal. Rare, use only when crowd positioning data is available (CFTC CoT, AAII, fund flows).

## 5 Detection Tools

### Tool 1: "So What" Test
Ask "So what?" three times on each first-order chain:
```
Flu season → Healthcare stocks ↑
→ So what? Flu happens every year, market has priced in seasonality
→ So what? Unless this year's flu is WORSE than expected (CDC data exceeds consensus)
→ So what? Real edge = monitoring CDC ILINet as leading indicator before data release
```

### Tool 2: "Priced In?" Checklist
- □ Is information public?
- □ What growth does current price imply?
- □ Have analyst estimates already adjusted?
- □ What does options market think (IV)?
- □ What does futures curve say?
If all = "Yes" → No edge. Look for second-order effects.

### Tool 3: "Hidden Winner" Search
For each chain's obvious winner, ask: Who is LESS OBVIOUS?
```
GLP-1 drugs explode → Obvious: LLY, NVO
Hidden winner: Weight loss success → clothes don't fit → clothing consumption ↑
Hidden loser: Eat less snacks → MDLZ↓; fewer bariatric surgeries → ISRG↓
```

### Tool 4: "Time Mismatch" Arbitrage
Markets overreact short-term, underreact long-term.
```
Short-term overreaction → Contrarian opportunity (hurricane → insurance crash → but insurers raise prices)
Long-term underreaction → Patience play (climate → grid upgrade, but stocks haven't moved)
Sweet spot: Lagged effects (1-3 weeks) — happening but not yet confirmed in data
```

### Tool 5: "Whose Expectations Are Changing?"
Not absolute good/bad, but direction of expectation change:
```
Bad company + extremely low expectations + marginal improvement = Explosive opportunity
Good company + sky-high expectations + any stumble = Crash risk
```

## Confidence Adjustment
| Finding | Adjustment |
|---------|-----------|
| Chain = consensus, no expectation gap | Downgrade |
| Chain has unique link market missed | Neutral |
| Chain found clear expectation gap | Upgrade |
| Found hidden winner/loser | May be the most valuable output |
