# Consensus Decode — 共识解码与认知偏差诊断

## Purpose
Before finding surprising connections, you must know what's NOT surprising. This skill maps the consensus and tags the cognitive biases that create it. The output feeds into BOTH the creative engine (what to avoid) and the validation engine (what counts as "already priced in").

## Step 1: Map First-Order Reactions

For any input, list the top 2-3 reactions that most people (especially retail investors) would have:

```
Input: "好多人感冒了"
  Reaction 1: 买药企 (JNJ, PFE)
  Reaction 2: 口罩/防疫概念
  Reaction 3: 远程办公 (ZM)
```

These are the consensus. They are your "do not go here" markers.

## Step 2: Tag Cognitive Biases

For each first-order reaction, identify WHY people think that way:

| Bias | Description | How to Detect | Example |
|------|------------|--------------|---------|
| **Availability** | Most easily recalled = assumed most likely | "This is the first thing that comes to mind" | 感冒→药企 (药is the most available association) |
| **Anchoring** | Locked onto one frame, ignoring alternatives | "Everyone is thinking within [X] frame" | 生病→anchored on "医疗" frame, missing "行为变化" frame |
| **Narrative Fallacy** | Good story ≠ good logic | "This sounds like a compelling story but..." | 疫情→口罩暴涨 is a good story, but regular flu ≠ pandemic |
| **Survivorship Bias** | Only remembering winners | "People remember the time X worked, not the times it didn't" | 只记得流感季药企涨的案例，忽略不涨的 |
| **Temporal Compression** | Mixing different time scales | "Treating a 2-week event as a structural shift" | 流感是2-4周事件，不会改变药企基本面 |

## Step 3: Check Known Patterns

Does this input map to a studied financial phenomenon?

### Pattern Library
| Input Signal | Known Pattern | Academic Source | Status |
|-------------|--------------|----------------|--------|
| 周一情绪低迷 | Monday Effect | Cross (1973), French (1980) | Largely arbitraged away in large-caps; persists in small/retail-heavy stocks |
| 一月份了 | January Effect | Wachtel (1942) | Small-cap outperformance in January; weakened but not eliminated |
| 五月了 | Sell in May | Bouman & Jacobsen (2002) | Nov-Apr outperforms May-Oct historically; inconsistent |
| 假期前/后 | Holiday Effect | Ariel (1990) | Pre-holiday returns tend to be higher; small effect |
| 选举年 | Presidential Cycle | Stovall, Hirsch | Year 3 of cycle historically strongest; sample size concerns |
| 流感季 | Flu → Absenteeism → Consumer behavior | Various | Well-documented CDC→behavior link, NOT well-priced by markets |
| 极端天气 | Weather → Energy/Agriculture | EIA, USDA | Weather premium in commodities is short-lived unless structural |

When a known pattern matches:
- **Acknowledge it** — don't pretend you invented the wheel
- **Go beyond it** — "Monday Effect 大家都知道，但你可能不知道的是..."
- **Check if it's still valid** — many calendar effects have been arbitraged away

## Output to Other Components

This skill produces a "consensus map" that feeds into:
1. **Connection Generator**: "These are consensus → avoid or go beyond them"
2. **Reality Check**: "If your connection matches consensus → downgrade"
3. **Insight Framing**: "Use the consensus as your Setup before the Flip"
