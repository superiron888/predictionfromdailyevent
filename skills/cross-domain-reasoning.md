# Cross-Domain Reasoning — 10-Discipline Financial Mapping

## Purpose
Maps 10 academic disciplines to financial market implications with bridging rules and quantitative anchors.

---

## Discipline Quick Reference

### 1. Psychology → Market Behavior
| Bias | Trigger | Financial Impact |
|------|---------|-----------------|
| Loss Aversion | Bad news/Black swan | Risk-off → GLD/TLT↑, SPY↓ |
| Herding | Social media/Meme | Retail influx → High volatility |
| Anchoring | Earnings miss/beat | Expectation gap drives price |
| FOMO | AI/Crypto hotspots | Late retail → Bubble signal |
| Recency Bias | Last crisis memory | Overreaction to similar events |

**Maslow → Economic Cycle**: Physiological needs (KO,PEP,XLU) = any economy → Safety (GLD,TLT) = downturn → Social (MCD,NFLX) = mid → Esteem (NKE,TSLA) = growth → Self-actualization (Luxury) = boom

**Anchors**: VIX >30 = panic, <12 = overconfidence. AAII spread >+30% = contrarian bearish, <-30% = contrarian bullish.

### 2. Physiology → Healthcare
| Signal | Scale Indicator | Industry | Anchors |
|--------|----------------|----------|---------|
| Flu/Cold | CDC ILINet >3.5% = strong season | Antiviral, OTC pharma | US flu healthcare ~$11B/yr, severe $20B+ |
| Obesity | CDC stats | GLP-1, Fitness | GLP-1 growth >50%/yr, penetration <5% |
| Mental health | NIMH reports | Telehealth, pharma | Growing post-COVID |

**Seasonal cycle**: Winter = flu (pharma + indoor media) → Spring = allergy + outdoor → Summer = travel/beverage → Fall = vaccine rollout

### 3. Physics/Energy → Commodity
```
HDD deviation +10% → Natural gas ~5-8% volatility
CDD deviation → Electricity demand ↑~2% per CDD above average
Hurricane Cat 3+ Gulf Coast → Oil +5-15% short term
```

### 4. Economics/Macro → Sector Rotation
| Signal | Direction | Benefits | Hurts |
|--------|-----------|----------|-------|
| CPI↑ | Inflation | XLE, GLD, TIP | TLT, QQQ |
| Weak NFP | Recession fear | TLT, QQQ (rate cuts) | KRE, XLY |
| Fed rate cut | Easing | QQQ, XLRE, GLD | KBE |

**Fed Reaction**: High inflation + strong jobs = hawkish. Low inflation + weak jobs = dovish. Goldilocks = best for stocks.

### 7. Geopolitics → Safe Haven + Supply
```
Event → Wave 1 (0-48h): VIX↑, GLD↑, TLT↑
     → Wave 2 (1-4wk): Commodity supply shock
     → Wave 3 (1-6mo): Inflation transmission
     → Wave 4 (6mo+): Supply chain restructuring
```
**Key**: 80% of geopolitical events revert in 1-4 weeks unless causing real supply disruption.

### 8. Supply Chain → Bottleneck Analysis
```
Bottleneck link = Maximum pricing power = Maximum profit elasticity
No alternative (single source) = High risk + High premium
Semiconductor cycle: ~3-4 years. Inventory/revenue >1.5x = excess, <0.8x = shortage
```

## Discipline Bridging Matrix

| From → To | Key Question |
|-----------|-------------|
| Psychology → Economics | How many people share this sentiment? Consumer confidence data? |
| Physiology → Economics | Prevalence? CDC data? Insurance coverage? |
| Physics → Economics | How many BTU/barrels/tons affected? |
| Meteorology → Economics | Impact area? Duration? USDA data? |
| Sociology → Economics | Trend or fad? Penetration rate? |
| Geopolitics → Economics | Which country controls what % of supply? |
| Supply Chain → Economics | How long can inventory last? Capacity recovery time? |
| Any → Market | Which transmission path? How fast? How large? Already priced in? |

---

## Cross-Validation Rules
- Same conclusion from 2 disciplines → confidence boost
- Historical precedent exists → confidence boost
- Counter-argument has easy failure conditions → confidence drop
- Conclusion = Bloomberg/CNBC headline → already priced in, confidence drop
